user signup and login with username and password 
in the profile page of user user clicks the cart info and request for booking 
then the user logout from the profile page because all information will be send to users email 
when the user forget the password enter the email that registered and forgetpassword link will be sent to the user email address. 
then by clicking the link user resets the password
in the admin page admin confirm the reservation, when the admin confirm to the reservation temporary id card will be sent to user by the email
